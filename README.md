# TraduDelta
A translation tool for Deltarune (PC)
```
TraduDelta.exe [MODE] [FILE/DIRECTORY]

--applymods                  Apply TS mods.
--json2po                    Convert .json to .po
--extracttext                Extract the texts from the Data.win
--clean                      Clean files.
```
